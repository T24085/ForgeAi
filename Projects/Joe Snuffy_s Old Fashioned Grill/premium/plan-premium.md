# Plan (Joe Snuffy_s Old Fashioned Grill - Premium)

## Summary
Premium, high‑end website plan for Joe Snuffy’s Old Fashioned Grill featuring restrained typography, warm color palette, subtle motion, and conversion‑focused sections. The design uses two carefully chosen web fonts, a clear typographic scale, and modern CSS patterns to deliver a polished, fast, and accessible experience.

- Menu
- Reservations
- About

- Our Story

## Style Notes
Fonts: Display – Playfair Display (serif, weight 700), Body – Inter (sans‑serif, weight 400). Typographic scale (6 steps): H1 – 96 px, H2 – 72 px, H3 – 48 px, H4 – 32 px, Body – 18 px, Caption – 14 px. Color story: #1C1C1C (charcoal), #D4A63F (warm amber), #F5F5F5 (neutral off‑white). Contrast: minimum 4.5:1 for body, 7:1 for headings. Spacing rhythm: base 8 px grid, use multiples (8, 16, 24, 32). Motion style: 200 ms ease‑in‑out for hover lifts, 400 ms for scroll‑triggered fades, micro‑animations only on interactive elements.
